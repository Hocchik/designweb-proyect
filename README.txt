# 🌐 React + Vite + Tailwind Starter

Este proyecto es una base moderna para aplicaciones frontend usando **React**, **Vite** y **Tailwind CSS**. Ideal para desarrollos rápidos, responsivos y con una estructura eficiente ⚡

## 🚀 ¿Qué incluye?

- ⚛️ React 18
- 🧩 Vite para bundling ultrarrápido
- 🎨 Tailwind CSS 3.x completamente configurado
- 🎯 Estructura lista para escalar con componentes

---

## 📦 Instalación

1. Clona el repositorio o inicia con esta plantilla:
   git clone https://github.com/tu-usuario/tu-repo.git
   cd tu-repo

2. 	Instala dependencias:
   npm install

3. 	Inicia el servidor de desarrollo:
   npm run dev

4. 	Abre en tu navegador:
http://localhost:5173
